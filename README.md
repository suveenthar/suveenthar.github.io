# Suveenthar T — Portfolio

🔗 **Live site:** [suveenthar.github.io](https://suveenthar.github.io)

My personal portfolio showcasing AI projects, Unity games, and full-stack web work.

## ✨ Features

- Dark, red & black themed UI with smooth animations
- Typewriter hero text, animated stat counters, scroll reveals
- Embedded Unity WebGL games (play directly in the browser)
- Project showcase with live demo + GitHub links
- Fully responsive, mobile-friendly design

## 🛠️ Built With

- HTML5, CSS3, JavaScript (vanilla)
- Font Awesome & Google Fonts
- Unity 6 / C# (for embedded games)

## 📂 Structure

```
├── index.html      # Main page
├── styles.css      # Styling
├── script.js       # Animations & interactions
└── assets/         # Images, icons, favicon
```

## 🚀 Run Locally

```bash
git clone https://github.com/suveenthar/suveenthar.github.io.git
cd suveenthar.github.io
```

Open `index.html` in your browser — no build step needed.
